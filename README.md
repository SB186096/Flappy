# FlappyFish

Exercise - Learn Flappy swim!

### Start the game

``` sh
git clone https://github.com/SB186096/Flappy.git
cd Flappy
pip install virtualenv
```

Create a virtualenvironment and source it:

``` sh
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
```

``` sh
python flappy_fish
```
